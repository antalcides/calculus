Section1.1Velocidad¶ permalink

El movimiento es frecuentemente modelado usando el cálculo. El concepto de velocidad media se utiliza para construir un modelo matemático. La velocidad media se define como el desplazamiento neto dividido por el tiempo transcurrido. Más precisament,e

Definition1.1.1Average Velocity

If \(p\) is a position function for something moving along a numbered line, then we define the average velocity over the time interval \(\cinterval{t_0}{t_1}\) to be: \begin{equation*}\frac{\fe{p}{t_1}-\fe{p}{t_0}}{t_1-t_0}\text{.}\end{equation*}

Subsection1.1.1Exercises

According to simplified Newtonian physics, if an object is dropped from an elevation 1 If you encounter an unfamiliar unit of measure while reading this lab manual, see Appendix D. of \(200\) meters (m) and allowed to free fall to the ground, then the elevation of the object is given by the position function \begin{equation*}\fe{p}{t}=200\,\text{m}-\left(4.9\,\sfrac{\text{m}}{\text{s}^2}\right)t^2\end{equation*} where \(t\) is the amount of time that has passed since the object was dropped. (The “200 m” in the formula represents the elevation from which the object was dropped, and the “4.9 m⁄s2” represents one half of the acceleration due to gravity near the surface of Earth.)
