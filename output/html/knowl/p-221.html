There is a new replacement law that can only be applied when \(x\to\infty\) or \(x\to-\infty\); this is Limit Law R3. Limit Law R3 essentially says that if the value of a function is increasing without any bound on how large it becomes or if the function is decreasing without any bound on how large its absolute value becomes, then the value of a constant divided by that function must be approaching zero. An analogy can be found in extremely poor party planning. Let's say that you plan to have a pizza party and you buy five pizzas.
